Q: What is WCF (Windows Communication Foundation) and why? Ans: WCF is a framework for building service-oriented application (SOA). Rather than requiring different technologies for different communication styles (like .NET Remoting for inter-process communication, MSMQ to communicate across heterogeneous networks, ASMX Web Service for HTTP as SOAP), WCF provides a single unified solution. Q: What is Endpoint? Ans: The endpoint is a fusion of address, contract, and binding. Every endpoint must have all three ABC elements where 'A'ddress defines where the service is, 'B'inding defines how to communicate and 'C'ontract defines what services does.
